At Home: Getting the Workshop Lab as a local VM

  • You need Vagrant (2.1.0+) and VirtualBox (5.2.12+)

  • Run the following commands to start the local VM:

# Getting the VM template (1 time only)
vagrant box add jeudi-mons-workshop https://github.com/
# Initialize the environment (1 time only)
mkdir ~/jeudi-mons-workshop && cd ~/jeudi-mons-workshop
vagrant init -m -f jeudi-mons-workshop
# Start the lab
vagrant up

  • Open the following URL: http://localhost:80

  • Welcome to the Lab Home Page!

On-Site: Getting the Workshop Lab as a Cloud VM

  • We provide you a set of Cloud VMs: only web browser is required

  • Access the "Instance Sheet Page", put your name on the "Name" column to allocate the instance for yourself, and click on the "Instance URL" link on the same line.

    • Instance sheet page

  • Welcome to the Lab Home Page!

Accessing Customized Slides

  • Each instance hosts a copy of this slide-deck, with customized links

  • If you click the link "Workshop Slides", a new tab will open, to this slide-deck

Accessing the "DevBox"

  • Each instance provides you a command line in HTML5

    • To access it, click the link "Devbox": DevBox direct Link

  • This "DevBox" contains the following tools:

    • Bash prompt

    • A local Docker engine

    • Java 8 (OpenJDK) and Maven 3

    • Vim/Nano/emacs for editing text files

Accessing Source Code

  • Each instance hosts a local git server.

  • Access the Gitea git server:

    • http://localhost:80/gitserver

    • Log in as the user butler (password is the same)

  • Access the source code:

    • Once authenticated, click on Explore (on top)

    • Click on butler/demoapp

      • Direct link: http://localhost:80/gitserver/butler/demoapp

Accessing Jenkins

  • Access your Jenkins instance:

    • http://localhost:80/jenkins

    • Log in as the user butler (password is the same)

    • This is the "Jenkins Classic GUI"
